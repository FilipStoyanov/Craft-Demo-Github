# Craft-Demo-Github

# Instructions
### Clone the repo
Git: 
`git clone https://github.com/FilipStoyanov/Craft-Demo-Github.git`

### Running the program
- Go to project folder
- Run `npm install`
- Run `npm start`
- Then enter GitHub user login (username) and Freshdesk subdomain

### Running the tests
- Run `npm run test`
