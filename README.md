# Craft-Demo-Github

# Instructions
### Clone the repo
Git: 
`git clone https://github.com/FilipStoyanov/Craft-Demo-Github.git`

### Running the program
- Go to project folder
- Run `npm install`
- Then run `npm start [GitHub user login (username)] [Freshdesk subdomain]` 

### Running the tests
- Run `npm run test`
